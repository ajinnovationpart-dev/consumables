## 시작하기

### 접속 방법

1. 관리자로부터 받은 웹 앱 URL 클릭
2. 로그인 페이지에서 **Google 계정으로 로그인** 버튼 클릭
3. Google 계정 선택 (여러 계정이 있는 경우)
4. 권한 승인 (최초 1회)
5. 등록된 사용자만 접근 가능

### 화면 구성

- **신청자**: 신청 폼 및 내 신청 현황
- **관리자**: 전체 신청 목록 및 관리 기능

## 신청자 가이드

### 신규 신청하기

#### 1단계: 기본 정보 입력

1. **품명** 입력 (필수)
   - 예: 연료필터, 브레이크패드, 에어필터

2. **수량** 입력 (필수)
   - 숫자만 입력 (최소 1)

3. **관리번호** 입력 (필수)
   - 자산 관리 번호

#### 2단계: 추가 정보 입력 (선택)

- **모델명**: 부품 모델명
- **시리얼번호**: 시리얼 번호
- **지역**: 드롭다운에서 선택
- **수령지**: 수령할 장소
- **전화번호**: 연락처
- **업체명**: 관련 업체명
- **비고**: 기타 참고사항

#### 3단계: 사진 첨부

1. **사진 첨부** 버튼 클릭
2. 사진 선택 (최대 5MB, JPG/PNG)
3. 미리보기 확인
4. 필요시 다른 사진 선택

#### 4단계: 신청 제출

1. 모든 필수 항목 입력 확인
2. **신청하기** 버튼 클릭
3. 신청번호 확인 (자동 생성)
4. 완료 메시지 확인

### 내 신청 현황 조회

#### 신청 목록 보기

- **내 신청 현황** 섹션에서 모든 신청 내역 확인
- 최신순으로 정렬
- 상태별 색상 표시

#### 상태 확인

- **접수중**: 신청 접수 완료, 처리 대기
- **접수완료**: 관리자가 접수 확인
- **발주진행**: 발주 진행 중
- **발주지연**: 발주 지연 (3일 이상)
- **발주완료**: 발주 완료, 수령 대기
- **처리완료**: 수령 확인 완료
- **접수취소**: 신청 취소됨

#### 신청 취소

1. **접수중** 상태인 신청만 취소 가능
2. 해당 신청의 **취소** 버튼 클릭
3. 확인 메시지에서 **확인** 클릭

#### 수령 확인

1. **발주완료** 상태인 신청만 수령 확인 가능
2. 해당 신청의 **수령확인** 버튼 클릭
3. 확인 메시지에서 **확인** 클릭
4. 상태가 **처리완료**로 변경됨

### 이메일 알림

신청 상태가 변경되면 자동으로 이메일 알림 발송:
- 접수완료
- 발주진행
- 발주지연
- 발주완료
- 접수취소

## 관리자 가이드

### 전체 신청 목록 조회

#### 기본 화면

- 모든 신청 내역이 테이블로 표시
- 최신순으로 정렬
- 상태별 색상 표시

#### 필터 사용

1. **상태** 필터: 특정 상태만 조회
2. **지역** 필터: 특정 지역만 조회
3. **시작일/종료일**: 기간별 조회
4. **필터 적용** 버튼 클릭
5. **초기화** 버튼으로 필터 해제

### 신청 상세 보기

1. 신청 목록에서 **상세** 버튼 클릭
2. 모달 창에서 상세 정보 확인:
   - 신청자 정보
   - 부품 정보
   - 사진 확인
   - 처리 이력

### 상태 변경

#### 개별 상태 변경

1. 신청 목록에서 **상세** 버튼 클릭
2. 모달에서 상태 변경 (향후 기능)

#### 일괄 상태 변경

1. 변경할 신청 건 체크박스 선택
2. 하단 버튼 클릭:
   - **접수완료**: 선택 건을 접수완료로 변경
   - **발주진행**: 선택 건을 발주진행으로 변경
   - **발주완료**: 선택 건을 발주완료로 변경
   - **취소**: 선택 건을 취소로 변경
3. 확인 메시지에서 **확인** 클릭

### 담당자 배정

1. 배정할 신청 건 체크박스 선택
2. **담당자 배정** 버튼 클릭
3. 담당자 이메일 입력
4. **확인** 클릭

### 사진 확인

1. 신청 목록에서 **보기** 버튼 클릭
2. 새 탭에서 사진 확인
3. Drive 링크로 접근

## 자주 묻는 질문

### Q1: 사진을 첨부하지 않으면 어떻게 되나요?

**A**: 사진 첨부는 필수 항목입니다. 사진 없이는 신청을 제출할 수 없습니다.

### Q2: 신청 후 수정할 수 있나요?

**A**: 신청 후에는 직접 수정할 수 없습니다. 취소 후 재신청하거나 관리자에게 문의하세요.

### Q3: 신청번호는 어떻게 생성되나요?

**A**: 자동으로 생성됩니다. 형식: `YYMMDD0001` (연월일 + 순번)

### Q4: 이메일 알림을 받지 못했어요

**A**: 
1. 스팸 폴더 확인
2. 이메일 주소 확인
3. 관리자에게 문의

### Q5: 사진 크기 제한이 있나요?

**A**: 최대 5MB까지 첨부 가능합니다. JPG 또는 PNG 형식만 지원됩니다.

### Q6: 모바일에서도 사용할 수 있나요?

**A**: 네, 모바일 브라우저에서도 사용 가능합니다. 반응형 디자인으로 최적화되어 있습니다.

### Q7: 신청을 취소할 수 있나요?

**A**: **접수중** 상태인 신청만 취소할 수 있습니다. 다른 상태에서는 관리자에게 문의하세요.

### Q8: 발주 완료 후 어떻게 하나요?

**A**: **수령확인** 버튼을 클릭하여 수령을 확인하세요. 상태가 **처리완료**로 변경됩니다.

### Q9: 관리자는 누구인가요?

**A**: **사용자관리** 시트에서 역할이 **관리자**로 설정된 사용자입니다.

### Q10: 시스템에 접근할 수 없어요

**A**: 
1. Google 계정으로 로그인되어 있는지 확인
2. 사용자 등록 여부 확인 (관리자에게 문의)
3. 웹 앱 URL이 올바른지 확인

## 문제 해결

### 로그인 문제

- Google 계정 확인
- 사용자 등록 여부 확인
- 관리자에게 문의

### 신청 제출 실패

- 필수 항목 입력 확인
- 사진 첨부 확인
- 인터넷 연결 확인
- 페이지 새로고침 후 재시도

### 사진 업로드 실패

- 파일 크기 확인 (5MB 이하)
- 파일 형식 확인 (JPG/PNG)
- 인터넷 연결 확인

### 상태 변경 불가

- 권한 확인 (관리자만 가능)
- 상태 확인 (일부 상태는 변경 불가)
